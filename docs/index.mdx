# Docgen

> Simple and Rapid React Documentation

- :stuck_out_tongue: MDX & MD Syntax oriented.
- :tornado: Fast HMR
- :chart: Out of the Box Api & Component Props Table Generation
- :heart: Emoji Support https://github.com/wooorm/emoticon/blob/HEAD/support.md

## Get Started

```bash
yarn add @alife/cnfe-docgen -D
```

## Usage

```sh
./node_modules/.bin/docgen start
```

or add to your package.json file

```json
{
  "scripts": {
    "docs:start": "docgen start",
    "docs:build": "docgen build"
  }
}
```

## Dependencies

- Vite
- MDX
- react-docgen-typescript
