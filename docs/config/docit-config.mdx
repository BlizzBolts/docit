# Config

## docRoot

- type: `String`
- default: `./docs`

a relative path to current processing direction where locates all the documents.

## themeConfig

### nav

- type: `Array`
- default: `[]`

external/internal links that place on the header bar.

## siteConfig

### title

- type: `String`
- default: `Docit`

The title of your site.

### description

- type: `String`
- default: `Site Description`

a brief description about your site.

- defaults: `[]`
