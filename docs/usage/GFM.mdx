# GFM

GFM 是 GitHub Flavored Markdown 的缩写，也是一种 Markdown 标准（GFM Spec）

原文链接: https://github.github.com/gfm/

GFM 在原有 Markdown 语法的基础上添加了表格、CheckBox、列表等新语法

## 示例

### 表格

```md
| First Header | Second Header |
| ------------ | ------------- |
| Content Cell | Content Cell  |
| Content Cell | Content Cell  |
```

| First Header | Second Header |
| ------------ | ------------- |
| Content Cell | Content Cell  |
| Content Cell | Content Cell  |

### Lists

```
- Item A
- Item B
- Item C
```

- Item A
- Item B
- Item C

### 外部链接

```md
https://github.com/

[Github](https://github.com/)
```

https://github.com/

[Github](https://github.com/)
