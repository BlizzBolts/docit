# Sidebars

Docit will use system file structure as sidebars.

```
├── FAQ.mdx
├── feature-request.mdx
├── index.mdx
└── usages
   ├── static-resources.mdx
   ├── using-components-in-docit.mdx
   └── configs
      ├── index.mdx
      └── sidebars.mdx
```
